# G4Flex
## Atividade 01
### Criar um CRUD "Fala, Torcedor!"
- O CRUD deve conter duas tabelas:
  - Tabela "Equipes" com os atributos:
    - Nome
    - Série (Divisão do clube)
    - Quantidade de sócios-torcedores
    - Planos de sócio (diferentes categorias, devem existir 3 categorias diferentes)
  - Tabela "Torcedor" com os atributos:
    - Nome
    - CPF
    - Data de Nascimento
    - Time
    - Plano de sócio assinado (o mesmo da tabela "Equipes")
